# Traffic Jam Game
A traffic Jame game in NY
## Installing

    git clone https://github.com/thristram/trafficNY.git
    cd trafficNY
    npm install
    node ./bin/www

## Run Simulation

    http://localhost:3001
    
Note only 1 simulation can be run at a time
    
## Control
    http://localhost:3001/control
